📝 변경 사항 요약

🏡 신혼집 계산기 4가지 변경
① 히어로 카드에 예상대출액 + 월상환액(이자) 칩 추가 (매매·전세·월세 각각)
② 단계별 준비 가이드 카드 (5단계 진행상황 시각화)
③ 지금 할 일 팁 카드 (현재 단계 기반 동적 제안)
④ 입력 완성도 기반 단계 자동 감지 (지역·금액·소득 입력 여부)
✈️ 신혼여행 계획 4가지 변경
① 여행지 · 출발·귀국일 입력 카드 + D-Day 카운트다운
② 카테고리별 예산 배분 분석 카드 (항공/숙박/식비/액티비티/쇼핑)
③ 예약·준비 체크리스트 카드 (6개 핵심 항목)
④ 일정 카드 뷰 / 테이블 뷰 전환 기능 (모바일 편의성)
📊 요약 대시보드 2가지 추가
① 신혼여행·신혼집 상태 요약 카드 (비용 + 진행률 미니 바)
